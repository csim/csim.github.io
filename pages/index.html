About

In the novel, The Strange Case of Dr. Jeykll and Mr. Hyde, Mr. Poole is Dr. Jekyll’s virtuous and loyal butler. Similarly, Poole is an upstanding and effective butler that helps you build Jekyll themes. It’s made by @mdo. There are currently two themes built on Poole: Hyde Lanyon Learn more and contribute on GitHub. Setup Some fun facts about the setup of this project include: Built for Jekyll Developed on GitHub and hosted for free on GitHub Pages Coded with Sublime Text 2, an amazing code editor Designed and developed while listening to music like Blood Bros Trilogy Have questions or suggestions?
